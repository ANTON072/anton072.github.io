IPアドレスから都道府県を調べる

  • Node
  • JavaScript

August 11, 2018

業務でIPアドレスから都道府県を抽出して、都道府県別のランキングを表示する必要があったのでその方法をメモ。

このサイトからも調べられますが、IPアドレスからはざっくりとした住所は簡単に調べることができます。City か State/Region という項目なんですが、これがかなり曖昧で、東京でも「Tokyo」や「Nerima」などいくつも存在するので「都道府県に分割する」という要件にはマッチしません。

どこどこJPというサービスを利用すれば、かなり正確にIPアドレスから都道府県情報を調べることができるらしいのですが、初期費用料が10万円とかなり高額でしたので見送りました。

いろいろ検索していると、MaxMindという会社さんが無償のIP地理位置情報データベースをご提供されていました。

GeoLite2 « MaxMind Developer Site

製品版は有料で、開発版よりは精度が高いらしいのですが、今回のは規模が小さいものだったので開発版を使うことにしました。上記ページからGeoLite2 Cityをダウンロードします。

これはあくまでデータベースなので細かな処理はプログラムを書く必要があります。MaxMindさんがGitHubで、このデータベースを利用して Look UpするNode.jsのライブラリをご提供してくださっていますので、それを利用させていただくことにしました。

あとはIPアドレスの文字列に以下の関数を実行していきます。

function geolocation(ip) {
  return new Promise(resolve => {
    maxmind.open('./GeoLite2-City.mmdb', (err, cityLookup) => {
      const result = cityLookup.get(ip);
      if (result.subdivisions && result.subdivisions[0]) {
        resolve(result.subdivisions[0].iso_code);
      } else {
        resolve(null);
      }
    });
  });
}

result.subdivisions[0].iso_code から都道府県コードが参照できるので、result.jsonから都道府県名とマッピングします。

精度の問題があるかもしれませんが、MaxMindさんのライブラリとデータベースを利用すると無料でIPアドレスから都道府県名を調べられて非常に便利でした。MaxMindさん、ありがとうございました！
